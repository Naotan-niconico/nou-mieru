# 岩手農業リフォーラム LP

農家さん向けの小さなDX・ソフトウェア代行サービスの静的ホームページです。

## 中身

- `index.html`
- `styles.css`

## 無料公開の候補

### GitHub Pages
1. GitHubで新規リポジトリを作成
2. この2ファイルをアップロード
3. Settings → Pages → Deploy from branch
4. main / root を選択
5. 数分後に `https://ユーザー名.github.io/リポジトリ名/` で公開

### Cloudflare Pages
1. GitHubにこのファイルを置く
2. Cloudflare PagesでGitHubリポジトリを接続
3. Framework preset: None
4. Build command: 空欄
5. Output directory: `/`
6. Deploy
