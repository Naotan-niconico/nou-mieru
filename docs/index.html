こんな悩みに

便利なサービスはある。でも、現場にはまだ「ちょっと面倒」が残ります。

毎回、出荷先ごとにExcelを作るのが大変
LINEで来た注文をあとから整理するのが面倒
作業日誌・販売記録・JA提出用の記録がバラバラ
既存サービスを入れたけど、設定や運用が続かない
家族や従業員との情報共有がうまくいかない
ホームページや予約フォームを作りたいけど手が回らない

できること

大きな農業SaaSではなく、農家さんごとの「最後の数十メートル」を整えます。

Excel・スプレッドシート改善

売上、出荷、作業記録、農薬・肥料使用記録などを入力しやすく、集計しやすい形にします。

フォーム・LINE連携

注文受付、予約受付、問い合わせ、収穫体験の申込などを、スマホから扱いやすくします。

既存サービスの導入補助

農業管理サービスの初期設定、圃場登録、使い方整理、運用ルール作りをサポートします。

小さなWebページ制作

農産物紹介、直売情報、SNSリンク、問い合わせフォームなど、必要な情報を1ページにまとめます。

進め方

まずは「何を作るか」ではなく、「何が面倒か」を聞きます。

  1. 相談普段の作業、困っている記録、紙・Excel・LINEの使い方を聞きます。
  2. 整理本当に楽になる部分だけを選び、小さく作れる形にします。
  3. 試作Googleフォーム、スプレッドシート、簡単なWebページなどで素早く形にします。
  4. 調整実際に使ってもらい、入力しやすさ・見やすさを直します。
  5. 運用必要に応じて月1回程度の見直しや修正を行います。
